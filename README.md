# 🔧 طريق الورشة (منوال الورش)

دليل ذكي موجه لتوثيق وتقييم سمعة ورش صيانة السيارات المحلية في مدينة الرياض (صناعية النسيم والسلي، صناعية مخرج 18، والصناعية القديمة)، قائم على الأمانة الفنية ودقة التشخيص والتعامل الميداني الفعلي.

---

## ✨ المميزات البرمجية والتصميمية الجديدة

1. **تطبيق ويب تقدمي (PWA)**:
   - دعم التثبيت المباشر على الهواتف الذكية (iOS & Android) مع أيقونة تطبيق مخصصة ومصممة باحترافية.
   - العمل بالكامل دون الحاجة للاتصال بالإنترنت (**Offline-First**) بفضل الـ Service Worker الذي يقوم بتخزين الأصول محلياً.

2. **بناء Tailwind CSS محلياً**:
   - تم استبدال الـ CDN الخارجي بنسخة مجمعة ومضغوطة محلياً بحجم **28 كيلوبايت فقط**، مما أدى إلى سرعة تحميل لحظية للصفحة وحذف الأنماط غير المستخدمة.

3. **التحميل عند الطلب (Lazy Loading) المتوافق محلياً**:
   - يتم تحميل بيانات الورش لكل منطقة ديناميكياً فقط عندما يقوم المستخدم باختيار المنطقة المعنية، مما يقلل من حجم تحميل الصفحة الأولي بنسبة 80%.
   - متوافق بالكامل مع التشغيل المحلي (بروتوكول `file://` بالنقر المزدوج على `index.html`) دون مواجهة أخطاء CORS بفضل تقنية حقن السكريبتات الديناميكية.

4. **مؤشر تحميل هيكلي (Skeleton Loader)**:
   - تمكين تجربة مستخدم عصرية وسلسة تعرض هيكلاً متحركاً للورش أثناء تحميل البيانات.

---

## 🛠️ طريقة التشغيل والتثبيت المحلي

### 1. التشغيل المباشر والسريع:
يمكنك تشغيل البرنامج فوراً وببساطة عن طريق فتح ملف `index.html` في متصفحك مباشرة بالنقر المزدوج (دون الحاجة لتنصيب أي خوادم).

### 2. التشغيل باستخدام خادم محلي (لتجربة ميزات PWA):
لتجربة ميزات التثبيت على الهاتف وتخزين البيانات أوفلاين محلياً، قم بتشغيل خادم ويب خفيف عبر سطر الأوامر في مجلد المشروع:
```bash
# تثبيت التبعيات أولاً
npm install

# تشغيل خادم محلي
npx http-server
```
ثم قم بفتح الرابط الموضح في المتصفح (مثال: `http://localhost:8080`).

## ⚙️ طريقة التطوير والتعديل على التنسيقات

إذا قمت بإضافة أي كلاسات تيلويند جديدة في ملفات الـ HTML أو الـ JS وترغب في إعادة بناء ملف التنسيقات المجمع:
```bash
# لبناء ملف التنسيق المضغوط لمرة واحدة
npm run build:css

# لمراقبة التغييرات والبناء التلقائي أثناء التطوير
npm run watch:css
```

## 📁 هيكلية المجلدات الرئيسية

```text
├── css/
│   ├── input.css        # ملف التنسيق المصدر (يحتوي على إعدادات تيلويند والقواعد المخصصة)
│   └── tailwind.css     # ملف التنسيق المجمع النهائي المضغوط
├── js/
│   ├── data-areas.js    # تعريف المناطق الجغرافية ومصفوفة الورش الرئيسية
│   ├── data-naseem-sulay.js # بيانات ورش صناعية النسيم والسلي
│   ├── data-exit18.js       # بيانات ورش صناعية مخرج 18
│   ├── data-old-industrial.js # بيانات ورش الصناعية القديمة
│   └── app.js           # محرك التشغيل والبحث والتحميل الديناميكي وتفعيل PWA
├── icons/               # أيقونات التطبيق المخصصة للـ PWA
├── index.html           # واجهة الدخول والعرض الرئيسية للتطبيق
├── manifest.json        # ملف تكوين PWA وتثبيت التطبيق على الجوال
├── sw.js                # الخدمة السحابية المحلية للتخزين والتشغيل دون إنترنت
├── package.json         # تكوين بيئة التطوير والاعتمادات البرمجية
└── tailwind.config.js   # إعدادات محرك تيلويند وبناء الأنماط المخصصة
```
